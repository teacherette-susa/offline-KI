Muss exakt so im OIDC-Anbieter hinterlegt sein
Standard ist http://localhost:3000
Einstellungen → Konto → API-Schlüssel in Open WebUI
Muss in LM Studio geladen und der Server gestartet sein

Einrichtung Qwen3-TTS

            
Qwen3-TTS läuft als separater lokaler Server:

            
pip install qwen-tts
qwen-tts-demo Qwen/Qwen3-TTS-12Hz-0.6B-CustomVoice --port 8000

            
GPU empfohlen. Apple Silicon: 0.6B nutzen (16 GB RAM). URL http://localhost:8000 in Einstellungen eintragen.

Alle Anfragen werden lokal verarbeitet. Die KI läuft auf dem Schulserver (LM Studio + Open WebUI). Keine Daten verlassen das lokale Netzwerk. Schülerdaten dürfen entsprechend den Datenschutzrichtlinien eingegeben werden, da die KI lokal läuft.
